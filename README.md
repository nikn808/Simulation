
# Симуляция 🌍

Консольная пошаговая симуляция 2D-мира, населённого травоядными и хищниками.

## Механики мира

- Травоядные (🐇) ищут траву (🌿).
- Хищники (🐺) выслеживают и атакуют травоядных.
- На карте присутствуют препятствия (🗿, 🌳), сквозь которые нельзя ходить, а съеденная трава постепенно вырастает заново.
- **Голод:** Каждое существо тратит сытость во время хода. Если она падает до нуля, существо начинает терять здоровье.
- **Размножение:** При наличии пар живых существ одного вида есть шанс появления потомства.
- У каждого существа есть своя скорость передвижения — быстрые ходят первыми.

## Настройка (config.py)

Все параметры симуляции гибко настраиваются в файле `config.py`. Вы можете изменить:
- Размеры карты
- Характеристики существ (здоровье, скорость, урон, трата сытости)
- Шанс размножения
- Скорость восполнения травы на поле
- Внешний вид (эмодзи) объектов

## Установка

### Требования
- Python 3.10+

### Запуск

```bash
# Клонировать репозиторий 
git clone https://github.com/nikn808/Simulation.git

# Перейти в папку проекта
cd Simulation

# Запустить симуляцию
python main.py
```
